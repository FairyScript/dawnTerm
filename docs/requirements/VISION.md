# dawnTerm 能力清单

> 最后更新：2026-07-07

## current（系统正在满足）

- **布局持久化**（`layout-persistence`）：保存面板布局，下次打开自动恢复

## planned（路线图已规划，尚未实现）

详见 `docs/compose/specs/2026-07-07-development-roadmap-design.md`

- **终端核心引擎**（P0）：xterm.js + PTY 本地终端
- **本地终端体验**（P1）：设置面板、命令面板、搜索、快捷键、会话快照
- **远程终端**（P2）：SSH 连接管理 + 终端会话 + SFTP 文件管理 + Telnet
- **插件体系**（P3）：插件 API、内置插件拆分、外部插件加载、主题扩展
